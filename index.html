About JUDERMA

JUDERMA is an Android application designed for image classification using TensorFlow Lite. It allows users to classify images based on predefined categories with high accuracy.

Features

  • Real-time image classification
  • Integration with TensorFlow Lite for efficient processing
  • User-friendly interface for capturing and selecting images

Installation

To get started with JUDERMA, follow these steps:

  1. Clone the repository: 
    git clone https://github.com/mohsinhassan617/JUDERMA.git
  2. Navigate to the project directory: 
    cd JUDERMA
  3. Open the project in Android Studio.
  4. Sync the project with Gradle files by clicking on the "Sync Project with Gradle Files" button in the toolbar.
  5. Build the project by selecting "Build" from the top menu and choosing "Make Project" or pressing Ctrl+F9.
  6. Run the application on an Android device or emulator by clicking the "Run" button (the green triangle) in the toolbar or pressing Shift+F10.
